🚇 羅馬地鐵路線

A線 (橙線): Battistini ↔ Anagnina

路線：Battistini → … → Ottaviano（梵蒂岡、聖彼得大教堂） → Lepanto（聖天使堡） → Flaminio（人民廣場） → Spagna（西班牙階梯） → Barberini（特雷維噴泉） → Repubblica → Termini（中央車站，可轉乘B線） → Vittorio Emanuele → Manzoni → San Giovanni（可轉乘C線） → … → Anagnina


重要站點: Ottaviano (梵蒂岡)、Spagna (西班牙階梯)、Termini (中央車站)

B線 (藍線): Rebibbia/Jonio ↔ Laurentina/EUR Fermi

路線：Laurentina → … → EUR Palasport → … → Basilica San Paolo → Garbatella → Piramide（真理之口） → Circo Massimo（帕拉丁山丘） → Colosseo（羅馬競技場、古羅馬廣場） → Cavour → Termini（中央車站，可轉乘A線） → Castro Pretorio → Policlinico → Bologna


重要站點: Colosseo (競技場)、Termini (中央車站)、Piramide

C線 (綠線): Pantano ↔ San Giovanni

較新路線，連接郊區

💡 提示: 主要使用 A線 (Ottaviano站-梵蒂岡) 和 B線 (Colosseo站-競技場)。兩線在 Termini 站交匯。

🚇 米蘭地鐵路線

M1線 (紅線): Sesto FS/Rho Fiera ↔ Bisceglie/Sesto 1° Maggio

路線：Sesto FS → Sesto Rondo’ ... → Pasteur → Loreto → Lima → Palestro → San Babila → Duomo（米蘭大教堂） → Cordusio → Cairoli → Cadorna ... → Rho Fiera


重要站點: Duomo (大教堂)、Cairoli (城堡)、Conciliazione

M2線 (綠線): Abbiategrasso/Assago ↔ Cologno Nord/Gessate

重要站點: Centrale FS (中央車站)、Cadorna (火車站)、Porta Genova

M3線 (黃線): San Donato ↔ Comasina

路線：Comasina → ... Maciachini → Zara → Sondrio → Centrale FS（米蘭中央車站） → Repubblica → Turati → Montenapoleone → Duomo（米蘭大教堂） → Missori → Crocetta → P.ta Romana → Lodi TIBB → ... San Donato


重要站點: Duomo (大教堂)、Montenapoleone (購物區)

M5線 (紫線): Bignami ↔ San Siro Stadio

連接市中心與聖西羅球場

💡 提示: 主要使用 M1線 (紅線) 和 M3線 (黃線)。Duomo 站是多線交匯的核心站。
